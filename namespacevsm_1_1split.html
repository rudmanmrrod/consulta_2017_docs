Split list into an `n`-length list of arrays.

:param ls: List to be split.
:type ls: list

:param n: Number of splits.
:type n: int

:returns: List of arrays whose length is 'n'.

**Examples**
>>> ls = [1,5,6,8,2,8]
>>> mp_split_ls(ls, 4)
[array([1, 5]), array([6, 8]), array([2]), array([8])]
